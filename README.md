# 毛球与小驯养师｜静态互动幻灯

这是一个可以在 Mac、PC、iPad 或手机随时打开的 GitHub Pages 课程仓库。AI使魔训练营只跳转到公开 URL；课件本体不依赖老师的 Mac、MSI 或 AI使魔平台。每课默认提供真实 AI 实战，外部工具可能需要网络、浏览器权限或已有账号；对应的本地预演保证工具受限时仍能继续讲课。

## 两个入口

- 三年级 60 分钟高阶版：`https://sydneygemstone-sudo.github.io/ai-shimo-courses/`
- 一二年级十课版：`https://sydneygemstone-sudo.github.io/ai-shimo-courses/full-course/`
- 一二年级教师排练版：`https://sydneygemstone-sudo.github.io/ai-shimo-courses/full-course/?teacher=1`
- 源码与版本历史：`https://github.com/sydneygemstone-sudo/ai-shimo-courses`

根入口先让老师选择年龄路线。三年级路线直接进入 16 张全屏互动幻灯；一二年级路线进入十课地图，每课固定 9 张幻灯。

## 课堂语法

每一屏只做一件事：

1. 毛球问一个问题；
2. 孩子先回答；
3. 下一屏再揭晓；
4. 毛球发一个任务；
5. 工具独占一整屏；
6. 用刚才的结果解释原因；
7. 换一个新情境再判断；
8. 毛球复述，孩子补正；
9. 一个出口题结束。

十课地图可打开“教师排练”，也可直接使用教师排练链接。十课共 90 页，每页都有三块独立提词：老师照着说、孩子现在做什么、怎样自然接到下一页。上课投影前可关闭；临时查看时点右上角“本页怎么讲”或按 `S`。左右方向键、底部按钮和左右滑动都可翻页。

## 十课 AI 实战映射

| 课 | 真实 AI 工具 | 现场成果 |
|---|---|---|
| 1 | AutoDraw | 记录第几笔开始猜中 |
| 2 | Teachable Machine 图像项目 | 训练水杯/铅笔分类，再改变光线测试 |
| 3 | Quick, Draw! | 比较普通画法和特殊画法 |
| 4 | Teachable Machine 声音项目 | 训练“喵/汪”，再用新叫声测试 |
| 5 | Blob Opera | 分清人的控制与模型自动和声 |
| 6 | Perplexity / ChatGPT / Gemini / Claude | 向 AI 提问并打开来源核查 |
| 7 | ChatGPT / Gemini / Claude | 只用虚构资料生成安全邀请卡 |
| 8 | Say What You See | 修改提示并比较前后结果 |
| 9 | ChatGPT / Gemini / Claude | 生成虚构计划并检查人类闸门 |
| 10 | Giga Manga | 生成一个角色、找错并修改一次 |

每课第 5 页默认打开“真实 AI 实战”。孩子先预测、打开工具做一轮，再回课件选择真实结果并解释；“本地预演 / 断网兜底”只用于备课、权限受限或外部工具临时不可用。

## 开课前检查

1. iPad 横屏打开两个在线入口。
2. 任意进入一课，试一次上一页、下一页、左右滑动和教师提词。
3. 从十课地图抽查将要教授的真实 AI 入口，确认能在新标签页打开并返回课程。
4. 涉及摄像头或麦克风时，只拍物品或录动物叫声；先由老师批准权限。
5. 临时断网，确认每课可切换到本地预演并继续完成概念练习。
6. 全屏投影时确认毛球、问题和按钮均在一屏内，不出现纵向讲义。

## 维护与发布

修改后运行：

```bash
node scripts/validate.mjs
```

推送到 `main` 后由 GitHub Actions 验证并由 GitHub Pages 发布。公开仓库只包含净化后的静态课程；不要复制原 AI-Shimo 工作树中的服务配置、日志、profile、memory 或缓存。

## 教学边界

- 毛球是故事角色，不是生命，也不是永远正确的权威。
- 不要求孩子上传脸、声音、真实姓名、地址或其他私人资料。
- 真实发送、付款、发布、账号与个人数据不进入儿童工具流。
- Agent 实战只生成虚构计划，不执行发送、购买、发布、登录或账号修改；本地页面只做闸门判断。
